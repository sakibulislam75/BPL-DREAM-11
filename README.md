## 🌍 Overview

**BPL DREAM-11** is an interactive fantasy cricket application that allows users to build a custom cricket team using a limited virtual budget.

Users can select players, manage coins, and build a balanced team while following strategic constraints like player roles and budget limits.

---

## ✨ Key Features

- 🏏 **Player Selection System**  
  Browse and select players from a dynamic list with detailed stats and pricing.

- 💰 **Budget Management**  
  Start with **50,000 coins** and strategically manage your spending.

- 👥 **Team Building**  
  Create a maximum of **6-player dream team**.

- 🔄 **Add / Remove Players**  
  Easily manage your selected squad in real time.

- 📊 **Live Updates**  
  Instant updates on remaining budget and team composition.

- 📱 **Responsive UI**  
  Fully optimized for mobile, tablet, and desktop.

- 🔔 **User Feedback System**  
  Toast notifications for actions like selection and errors.

---

## 🎮 How to Play

1. Start with **50,000 coins**
2. Browse available BPL players
3. Select players based on strategy
4. Manage your budget wisely
5. Build a maximum **6-player team**
6. Finalize your dream squad 🏆

---

## 🛠️ Game Rules

- 💰 Maximum Budget: **50,000 coins**
- 👥 Maximum Players: **6**
- 🎯 Each player has a unique cost
- 🧠 Team must be strategically balanced
- ❌ Cannot exceed budget limit
